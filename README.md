# Blockchain Laboratory Works

## Overview
This repository contains 6 laboratory works implementing blockchain concepts from scratch in Python:

- **Lab 01**: Basic blockchain with SHA-256 hashing and chain validation
- **Lab 02**: Proof of Work consensus mechanism with difficulty adjustment
- **Lab 03**: Smart contracts and transaction simulation with rollback capability
- **Lab 04**: IoT device access control on private blockchain (Ganache)
- **Lab 05**: Multi-node private blockchain network (Geth)
- **Lab 06**: Merkle tree integration for efficient transaction verification

## Technologies
- Python 3.9+
- Solidity (smart contracts)
- Web3.py
- Geth (Go Ethereum)
- Ganache

## Key Features
- Proof of Work mining with adjustable difficulty
- Merkle root computation and transaction proofs
- Account-based state management
- Smart contract interpreter
- IoT device simulation
- Private blockchain network setup
